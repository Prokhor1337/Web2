# Лабораторна робота 4-6: Фільмова база

Ця лабораторна робота реалізує веб-додаток для перегляду бази телевізійних шоу, використовуючи асинхронні запити до відкритого API `TVMaze`.

## Що робить програма

- Асинхронно завантажує список шоу з API за адресою `https://api.tvmaze.com/shows`.
- Відображає кожне шоу у вигляді зручної картки з постером, назвою, рейтингом та жанрами.
- Дозволяє фільтрувати шоу (шукати за назвою) через текстове поле пошуку в реальному часі.
- Дозволяє сортувати відображені результати:
  - В алфавітному порядку.
  - За рейтингом (від найвищого).
- Показує повідомлення про помилку, якщо дані не вдалося завантажити.

## Основні функції файлу `laboratory4-6.js`

- `fetchMovies()`
  - Виконує запит до API через `fetch` та `async/await`.
  - Перевіряє `response.ok` та перехоплює помилки через `try/catch`.
  - Після успішного отримання даних викликає функцію рендерингу `displayMovies()`.

- `displayMovies(movies)`
  - Приймає масив об'єктів шоу.
  - Використовує сучасний синтаксис (деструктуризацію `const { name, image, rating, genres } = movie`) та шаблонні рядки для генерації HTML.
  - Очищує контейнер `moviesContainer` та вставляє нові елементи за допомогою `insertAdjacentHTML`.

- `handleProcessData()`
  - Слухає зміни в полі пошуку (`searchInput`) та списку сортування (`sortSelect`).
  - Спочатку фільтрує оригінальний масив `allMovies` методом `filter()`.
  - Потім застосовує сортування `sort()` до відфільтрованого масиву.
  - Передає готовий масив у `displayMovies()`.

- `showError(message)`
  - Відповідає за маніпуляцію DOM для виведення тексту помилок.

## Структура HTML та UI

- Поле пошуку `<input id="searchInput">`.
- Випадаючий список сортування `<select id="sortSelect">`.
- Контейнер результатів `<div id="moviesContainer">`.
- Блок для відображення помилок `<div id="errorMessage">`.

## Додатково

Додаток стилізований за допомогою `laboratory4-6.css` (використовуються CSS-гріди/флексбокси для карток). Всі змінні оголошені з використанням сучасного стандарту ES6 (`let`, `const`), а також застосовуються стрілочні функції.
